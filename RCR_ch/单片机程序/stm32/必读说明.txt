>> LM+ -> 左侧步进电机正转
>> LM- -> 左侧步进电机反转
>> LM2 -> 左侧步进电机180度旋转(会自动回中, 防止拉扯电线)
>> LMF -> 左侧步进电机顺时针微调
>> LMR -> 左侧步进电机逆时针微调

>> LC+ -> 左侧爪子夹紧(气缸)
>> LC- -> 左侧爪子松开(气缸)

>> DM+ -> 下侧步进电机正转
>> DM- -> 下侧步进电机反转
>> DM2 -> 下侧步进电机180度旋转(会自动回中, 防止拉扯电线)
>> DMF -> 下侧步进电机顺时针微调
>> DMR -> 下侧步进电机逆时针微调

>> DC+ -> 下侧爪子夹紧(气缸)
>> DC- -> 下侧爪子松开(气缸)

>> AO+ -> 两个爪子同时增加
>> AO- -> 两个爪子同时松开
